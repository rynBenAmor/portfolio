    IOT Project

    Description
    Real-Time IoT Monitoring Project with ESP8266 is a smart solution utilizing Arduino Uno and DHT to measure temperature and humidity. The ESP8266 ensures connectivity to a web server via WebSocket, enabling real-time transmission of measured data. This implementation showcases advanced skills in embedded systems programming, web server configuration with ESP8266WebServer, and efficient communication via WebSocket. Using an HTML page, the data is visualized and analyzed, providing a comprehensive and innovative IoT monitoring solution.

    Moncef Bay

    Description
    Moncef Bay est une boutique en ligne e-commerce réalisée avec Django. Cette plateforme complète offre une expérience d'achat simple, garantit également la sécurité grâce aux systèmes d'authentification complets de Django, et propose également des fonctionnalités CRUD (Create, Read, Update, Delete).     Projet IoT

    Description
    Le projet de surveillance IoT en temps réel avec ESP8266 est une solution intelligente utilisant Arduino Uno et DHT pour mesurer la température et l'humidité. L'ESP8266 assure la connectivité à un serveur web via WebSocket, permettant la transmission en temps réel des données mesurées. Cette mise en œuvre met en avant des compétences avancées en programmation de systèmes embarqués, en configuration de serveur web avec ESP8266WebServer et en communication efficace via WebSocket. À l'aide d'une page HTML, les données sont visualisées et analysées, fournissant une solution complète et innovante de surveillance IoT.
